# 【内部向け】ルールをまとめてみた

## 前提
- 決まってないのも結構あります。なので提案歓迎してます。例えば、「こういうwebサービス使ったらどうか」「こういうルールを追加・変更・修正すればよくなるんじゃないか」など。
discordの質問・要望チャンネルかGitHubのdocumentsリポジトリにissueたてるとか(決めた方が良いかな)、提案してくれたら秒速で導入して、良くなるようにします。
